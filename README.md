# Commission Fee Task

## How to install
```
composer i
```

## How to use
1. run this command
```
php artisan serve
```

2. [Open home page link](http://127.0.0.1:8000) (http://127.0.0.1:8000)
3. Select your csv file from your computer
4. Click on calculate button

## Test
For running test please run below command
```
php artisan test
```

Also, you can run this command
```
composer test
```
